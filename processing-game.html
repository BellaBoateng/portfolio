A Day in a Life

An interactive narrative game exploring identity, bias, microaggressions, and privilege through choice-driven storytelling.

Role: Designer + Developer Tools: Processing, Java, Visual Design Type: Interactive Media Final Project

Overview

A game about perspective, identity, and hidden bias.

A Day in a Life is a Processing-based game where players move through everyday scenarios while their character identity is hidden. Through scenes involving moments like being followed, job searching, or navigating a classroom, the game asks players to reflect on how identity shapes everyday experiences.

Goal

“Turn interactive play into awareness, reflection, and dialogue.”

Reflection

What this project taught me.

This project pushed me to think about interaction as more than just functionality. I learned how game mechanics, visual design, pacing, and narrative structure can work together to create reflection. It also helped me practice translating sensitive social themes into an interactive format that feels engaging while still being intentional and thoughtful.
